# 工具 API 参考

Doctra 工具函数和辅助函数的文档。

## 概述

工具模块为常见任务提供辅助函数。

## 可用的工具

### 文件操作

- PDF I/O 操作
- 图像加载和保存
- 目录管理

### 边界框操作

- 坐标变换
- 框交集和并集
- 框过滤和排序

### 进度跟踪

- 进度条管理
- 状态报告

### OCR 工具

- 文本清理和规范化
- 语言检测辅助函数
